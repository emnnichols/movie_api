ghibliFLIX API

The server-side component of a web application that will provide users with access to information about Studio Ghibli films.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.


API Endpoints

REQUEST URL HTTP
METHOD		 REQUEST
BODY FORMAT		 RESPONSE
BODY FORMAT
getAllMovies /movies GET None Array of JSON objects with data for all movies, example:

[{
id: {number}, Title: {string}, Description: {string},
Genre: { Name: {string}, Description: {string} },
Director: { Name: {string}, Bio: {string}, Birth {string}, Death: {string} },
Released: {number}, ImagePath: {string},
Featured: {boolean}
},{...},{...}]
getMoviesByYear /movies/year/:released GET movie.released An array of JSON objects with data for movies from a specific year
getMoviesByGenre /movies/genres/:genre GET movie.genre.name An array of JSON objects with data for movies with a specific genre
getMoviesByDirector /movies/directors/:director GET movie.director.name An array of JSON objects with data for movies from a specific director
getSingleMovie /movies/:movieId GET movie._id JSON object with data for single movie, example:

{
id: {number}, Title: {string}, Description: {string},
Genre: { Name: {string}, Description: {string} },
Director: { Name: {string}, Bio: {string}, Birth {string}, Death: {string} },
Released: {number}, ImagePath: {string},
Featured: {boolean}
}
getGenre /movies/genres/:genre/about GET movie.genre.name A JSON object with data about a genre, example:

{
Name: {string},
Description: {string}
}
getDirector /movies/directors/:director/about GET movie.director.name A JSON object with data about a director, example:

{
Name: {string},
Bio: {string},
Birth: {string},
Death: {string}
}
userRegistration /signup POST A JSON object with new user data, example:

{
Username: {string, required},
Password: {string, required},
Email: {string, required},
Birthday: {date}
} 		A JSON object with new user data and an assigned ID, example:

{
id: {number},
Username: {string},
Password: {string},
Email: {string},
Birthday: {date},
FavoriteMovies: [ {movie}, {movie} ]
}
getUserByUsername /profile/:Username GET user.Username A JSON object with data about a user:

{
id: {number},
Username: {string},
Email: {string},
Birthday: {date},
FavoriteMovies: [ {movie}, {movie} ]
}
updateUser /profile/:Username/account PUT A JSON object with data to be updated, example:

{
"Email": {updated string}
} 		A JSON object with all user info with the updated data, example:

{
id: {number},
Username: '{string},
Password: {string},
Email: {updated string},
Birthday: {date},
FavoriteMovies: [ {movie}, {movie} ]
}
deleteUser /profile/:Username/account DELETE user.Username A message indicating the user has been deleted sucessfully
addFavMovie /profile/:Username/movies/:movieId POST user.Username, movie._id A JSON object of the user data, example:

{
id: {number},
Username: {string},
Password: {string},
Email: {string},
Birthday: {date},
FavoriteMovies: [ {movie}, {new movie} ]
}
deleteFavMovie /profile/:Username/movies/:movieId DELETE user.Username, movie._id A JSON object of the user data, example:

{
id: {number},
Username: {string},
Password: {string},
Email: {string},
Birthday: {date},
FavoriteMovies: [ {movie} ]
}
