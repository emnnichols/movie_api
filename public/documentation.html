Studio Ghibli API

Description

The server-side component of a web application that will provide users with access to information about Studio Ghibli films.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.


Supported Requests

Request URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object with data about all the movies
Get data about a single movie by name /movies/[title] GET None A JSON object with data about a single movie, example:
{
id: #,
title: 'movie name',
director: 'first last',
year: ####,
genre: 'genre'
}
Get a list of movies by year /movies/year/[year] GET None A JSON object with data about movies from a specific year
Get list of movies by genre /movies/genres/[genre] GET None A JSON object with data about movies with a specific genre
Get data about a genre /genres/[genre]/about GET None A JSON object with data about a genre, example:
{
id: #,
genre: 'genre',
description: 'desc'
}
Get list all genres /genres GET None A JSON object with data about all available genres
Get a list of movies by a director /movies/directors/[director] GET None A JSON object with data about movies from a specific director
Get data about a director /movies/directors/[director]/about GET None A JSON object with data about a director, example:
{
id: #,
name: 'first last',
birth: ####,
death: ####,
bio: 'bio'
}
Get a list of all users /users GET None A JSON object with data about all users
Add a user /users POST A JSON object with data about the user to be added, example:
{
username: 'username',
name: 'first last',
email: 'email',
movies: []
} 		A JSON object with data about the user added along with an ID, example:
{
username: 'username',
name: 'first last',
email: 'email',
movies: [],
id: ####
}
Update a username /users/[username] PUT None A text message indicating the username has been updated
Delete a user /users/[username] DELETE None A text message indicating the user has been deleted sucessfully
Add a movie to favorites /users/[username]/movies POST A JSON object with data about the movie to be added, example:
{
id: #,
title: 'movie name',
director: 'first last',
year: ####,
genre(s): {genre: 'genre'}
} 		A text message indicating the movie has been added
Remove a movie from favorites /users/[username]/movies/[title] DELETE None A text message indicating the movie has been removed from favorites list
