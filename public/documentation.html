Studio Ghibli API

Description

The server-side component of a web application that will provide users with access to information about Studio Ghibli films.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.


Supported Requests

Request URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None An array of JSON objects with data about all the movies, example:
[{
id: #,
Title: 'movie name',
Description: 'desc',
Genre: {Name: '', Description: ''},
Director: {Name: '', Bio: '', Birth '', Death: ''},
Released: ####,
ImagePath: 'image.jpeg',
featured: true/false
},{...},{...}]
Get data about a single movie by name /movies/:Title GET None A JSON object with data about a single movie, example:
{
id: #,
Title: 'movie name',
Description: 'desc',
Genre: {Name: '', Description: ''},
Director: {Name: '', Bio: '', Birth '', Death: ''},
Released: ####,
ImagePath: 'image.jpeg',
featured: true/false
}
Get a list of movies by year /movies/year/:Released GET None A JSON object with data about movies from a specific year
Get list of movies by genre /movies/genres/:Genre GET None A JSON object with data about movies with a specific genre
Get data about a genre /movies/genres/:Genre/about GET None A JSON object with data about a genre, example:
{
Name: 'genre',
Description: 'desc'
}
Get a list of movies by a director /movies/directors/:Director GET None A JSON object with data about movies from a specific director
Get data about a director /movies/directors/:Director/about GET None A JSON object with data about a director, example:
{
Name: 'first last',
Bio: 'bio',
Birth: 'MM DD, YYYY',
Death: 'MM DD, YYYY'
}
Get a list of all users /users GET None An array of JSON objects with data about all users
Get a user by username /users/:Username GET None A JSON object with data about a user:
{
id: #,
Username: 'username',
Password: 'password',
Email: 'email',
Birthday: date,
FavoriteMovies: [movies]
}
Add a user /users POST A JSON object with data about the user to be added, example:
{
Username: 'username [required]',
Password: 'password [required]',
Email: 'email [required]',
Birthday: date
} 		A JSON object with data about the user added along with an ID, example:
{
id: #,
Username: 'username',
Password: 'password',
Email: 'email',
Birthday: date,
FavoriteMovies: []
}
Update user info by username /users/:Username PUT A JSON object with the field and value to be updated, example:
{
"Username": "username",
"Password": "password",
"Email": "updatedemail@email.com"
} 		A JSON object with all user info with the updated field/value, example:
{
id: #,
Username: 'username',
Password: 'password',
Email: 'updatedEmail',
Birthday: date,
FavoriteMovies: [movies]
}
Delete a user by username /users/:Username DELETE None A message indicating the user has been deleted sucessfully
Add a movie to favorites /users/:Username/movies/:MovieID POST None A JSON object of the user with the updated movie list, example:
{
id: #,
Username: 'username',
Password: 'password',
Email: 'email',
Birthday: date,
FavoriteMovies: [ "favmovie1", "newFavorite" ]
}
Remove a movie from favorites /users/:Username/movies/:MovieID DELETE None A JSON object of the user with the updated movie list, example:
{
id: #,
Username: 'username',
Password: 'password',
Email: 'email',
Birthday: date,
FavoriteMovies: [ "favmovie1" ]
}
